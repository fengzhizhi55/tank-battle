# 坦克大战 (Tank Battle)

经典坦克大战游戏的 HTML/JavaScript 实现，纯前端，无需服务器即可运行。

## 玩法

- **目标**: 保护基地（右下角鹰旗），消灭所有敌方坦克
- **操作**:
  - `↑ ↓ ← →` 移动坦克
  - `Space` 射击
  - `P` 暂停/继续

## 特性

- 砖墙可被子弹摧毁
- 钢墙不可摧毁
- 敌方坦克 AI 自动追踪并射击
- 波次系统，每波敌人逐渐增多
- 玩家拥有 3 条命
- 爆炸动画效果

## 如何运行

直接用浏览器打开 `index.html` 即可开始游戏。

## 技术栈

纯 HTML + CSS + JavaScript (Canvas 2D)，零依赖。
